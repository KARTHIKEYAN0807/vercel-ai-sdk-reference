# Generating and Streaming Text

This directory contains TypeScript examples and utility code that demonstrate
the Vercel AI SDK’s text‑generation and streaming capabilities.

## Contents

- **`fullStream property.ts`** – shows how to access the `fullStream` field on
  a streaming response.
- **`genrate text advanced prompts.ts`** – advanced prompt‑engineering
  techniques when generating text.
- **`Lifecycle callbacks (experimental).ts`** – examples of lifecycle hooks
  (`onStart`, `onChunk`, `onFinish`, etc.).
- **`onError callback.ts`** – handling errors during a stream.
- **`onFinish callback.ts`** – post‑processing after a stream completes.
- **`Sources.ts`** – utilities for working with prompt sources.
- **`Stream transformation.ts`** – modifying or filtering chunks as they arrive.
- **`streamText.ts`** – basic streaming example.
