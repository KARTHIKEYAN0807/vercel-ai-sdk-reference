# Generating Structured Data

This directory contains TypeScript examples and utilities demonstrating how to
produce structured outputs with the Vercel AI SDK.

## Contents

- **`Accessing Reasoning.ts`** – shows how to retrieve reasoning data from a
  response.
- **`Error Handling in Streams.ts`** – handling errors while streaming
  structured data.
- **`Generating Structured Outputs.ts`** – basic example of producing a
  JSON‑structured response.
- **`Generating Structured Outputs with Tools.ts`** – combining tools with
  structured output generation.
- **`Property Descriptions.ts`** – defining schemas with detailed property
  descriptions.
- **`Stream Structured Outputs.ts`** – streaming scenario for structured data.
